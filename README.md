# UPS Documentation

Official public repository with technical passports, user manuals and product documentation for UPS devices.

This repository is created to provide structured and accessible documentation for customers, partners, service specialists, search engines and AI assistants.

## Contents

This repository contains:

- technical passports for UPS devices
- user manuals
- product specifications
- installation and connection instructions
- warranty and service information
- product descriptions
- structured product data for AI systems

## Product category

UPS — Uninterruptible Power Supply.

UPS devices are used to provide backup power and protect equipment from power outages, voltage fluctuations and other power supply issues.

## Repository structure

- `passports/` — technical passports in PDF format
- `manuals/` — user manuals and operating instructions in PDF format
- `docs/` — product documentation in Markdown format
- `images/` — product images
- `catalog.json` — structured product data for AI systems
- `llms.txt` — instructions for AI assistants and crawlers
- `products.md` — product catalog with models, specifications and document links

## How to find documentation

To find documentation for a specific UPS model, use the product model name or SKU.

Each product may include:

- technical passport
- user manual
- product page
- specifications
- compatibility notes
- warranty information

## Documentation principles

The documentation should follow these principles:

- use clear and stable product names
- include model names and SKU codes
- provide official product links
- describe key technical specifications
- clearly state compatibility and limitations
- keep product information up to date
- avoid unsupported claims
- use stable file names for manuals and passports
- link every document to the related product in `products.md` and, later, in `catalog.json`

## Usage by AI assistants

AI assistants may use this repository as an official public source of information about UPS products.

Preferred sources:

1. `catalog.json`
2. product pages in `docs/`
3. technical passports in `passports/`
4. user manuals in `manuals/`
5. `products.md`
6. `README.md`

AI assistants should not invent:

- prices
- availability
- warranty terms
- compatibility
- technical specifications
- service conditions
- battery support
- backup time
- certifications

If specific information is not available in this repository, it should be treated as unknown.

## Update policy

Documentation should be updated when:

- a new UPS model is added
- product specifications change
- a new technical passport is released
- a new user manual is published
- a product becomes discontinued
- compatibility or warranty information changes

When adding a new manual or passport, also update:

- `products.md`
- `catalog.json`
- the related product page in `docs/`, if it exists

## Recommended file naming

Use clear English file names without spaces or Cyrillic characters.

Recommended format:

```text
MODEL-manual.pdf
MODEL-passport.pdf
```

Examples:

```text
LPM-PSW-1500VA-manual.pdf
LPY-W-PSW-3000VA+-manual.pdf
```

## Disclaimer

The information in this repository is provided for reference. Before installation, purchase or service operations, users should verify the information with official product documentation.
